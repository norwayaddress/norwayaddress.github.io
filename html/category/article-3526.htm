/方式1

第一道菜：鸡（大吉大利）

电饭煲葱香手撕鸡食材调料：三黄鸡 1只 / 小葱 150克 / 花椒粒 10克 / 姜片 5片 / 老抽 3勺/ 生抽 3勺 / 料酒 2勺/ 糖 1勺/ 盐 适量

1、碗里放姜片，花椒粒，生抽，料酒，老抽，白糖，食盐，混合均匀腌制15分钟备用。

2、三黄鸡加入保鲜袋中，加入准备好的酱汁，袋子封口，抓匀后腌制2小时。

3、电饭锅底铺好葱段，放入腌制好的鸡，倒入酱汁，盖好盖子后按煮饭键。

4、时间到开盖装盘，美味的电饭煲葱香手撕鸡就可以开吃啦，除夕吃鸡，大吉大利。

步骤/方式2

第二道菜：鱼（年年有鱼）

清蒸桂鱼食材调料：桂鱼 600g / 蒸鱼豉油 30ml / 大葱 1段 / 胡萝卜 1段 / 小葱段 适量 / 姜片 适量

1、大葱去芯切丝，胡萝卜先切片再切丝，都切好后放清水中浸透备用。

2、买鱼的时候叫卖家把内脏处理干净，拿回来清洗，然后切波浪花刀，一面切四刀。

3、另一面从鱼中间连着鱼骨切到底，切到头也切到尾，不要切断。

4、切头切尾的那面往外翻，让鱼站立在盘中。

5、放上姜片和葱段，待锅里水烧开上汽，把鱼上锅大火蒸8分钟。

6、蒸好后开盖端出，拿掉葱姜，倒掉蒸鱼的水，然后放上准备好的大葱丝和胡萝卜丝。

7、从鱼的周围淋入蒸鱼豉油，再从葱丝上淋入热油开吃美味的清蒸桂鱼啦。

步骤/方式3

第三道菜：韭菜（长长久久）

凉拌韭菜食材调料：韭菜 300克 / 蒜蓉 20克 / 红油辣椒 适量 / 小米辣 少许 / 陈醋 2勺 / 生抽 2勺 / 鸡精 1勺/ 白糖 1勺 / 食盐 1勺

1、韭菜洗净切段，起锅烧几滴橄榄油，1勺食盐，下韭菜烫至变色捞出放碗里备用。

2、酱汁碗里下生抽，陈醋，小米辣，蒜蓉，鸡精，白糖，食盐和红油辣椒搅拌均匀。

3、把拌好的酱汁均匀淋在烫好的韭菜上拌匀就可以摆盘开吃啦。

步骤/方式4

第四道菜：腐竹（福气满满）

麻辣腐竹食材调料：腐竹300克 / 青红椒段适量 / 蒜瓣10克 / 老干妈油辣椒2勺 / 盐少许 / 生抽1勺 / 陈醋1勺 / 香油1勺 / 花椒粉适量 / 葱花适量 / 白糖1勺

1、泡发好的腐竹下锅焯水1分钟，加入少许盐，焯水好的腐竹放凉水里浸透。

2、青红椒段和蒜瓣一起放进拉蒜神器里，拉成碎末倒出装碗，再调入老干妈油辣椒。

3、接着调入生抽陈醋香油花椒粉和葱花，还有适量食盐和白糖拌匀成调味酱。

4、将凉水里浸透的腐竹捞出沥水装盘，最后放上调味酱汁，吃的时候拌匀就可以啦。

步骤/方式5

第五道菜：丸子（团团圆圆）

牛肉丸萝卜汤食材调料：牛肉丸300克 白萝卜1个 姜片3片 芹菜碎少许 葱花少许 食盐1克 酱油15毫升

1、海鲜锅中加适量清水，下过萝卜和姜片煮30分钟，加入牛肉丸继续煮5分钟。

2、调入盐和生抽调味，起锅前撒上芹菜碎和葱花。

3、美味的牛肉丸萝卜汤就做好啦，营养开胃，团团圆圆。

1、【虾仁蒸蛋】1.胡萝卜去皮后切小沫，虾仁洗净后切小丁；2.鸡蛋打匀后加水、微量盐、胡萝卜末、虾仁丁；3.蒸锅内蒸6—8分钟，蛋液凝固后取出淋些许香油。

2、【蚝油】南瓜去皮切块放入碗中，撒上蒜蓉；撒少许盐；加入少许鸡精；加入一大勺蚝油，搅拌均匀，高火加热6分钟左右即可。

3、【三汁焖花菜】1.花菜先用淡盐水，洗净后掰成小朵。锅里热油，放入花菜煸炒，加少许水闷一小会变软后，加入调味料装盘后撒上熟芝麻。

6、【酱爆火腿鲍菇】杏鲍菇洗净，切成小丁;黄瓜洗净，切丁;火腿切丁;洋葱洗净，切小丁;锅中倒入油适量，油温略高后转小火，放入洋葱、酱，翻炒;放入菇丁，翻炒;放入盐、白糖、鸡精、水，炒匀后加盖煮2分钟左右至菇丁熟;放入黄瓜、火腿，大火收汁，分两次淋入水淀粉勾好芡盛出即可。

7、【香辣豆腐皮】1.豆腐皮切条，入沸水中汆烫1、2分钟，捞出沥干水分2.起炒锅，热锅凉油爆香葱姜丝，下豆腐皮，翻炒均匀3.加入1小勺生抽，1/2小勺老抽，翻炒均匀4.下辣椒丝和香菜梗，加入盐调味，继续翻炒5.最后加入少许鸡精，关火，翻炒均匀，起锅即可。

8、【冰镇酸辣蜇头】1.海蜇清洗干净，放入清水中浸泡4小时，其间更换2—3次水;2.所有其他的材料清洗干净，洋葱切片，青红辣椒切碎、蒜切片、姜切末;3.取一只碗，碗里放入八角和花椒，加入小半碗的陈醋，加入一大勺糖，加入一勺的鲜酱油搅拌均匀，辅料放进去浸泡4小时;4.海蜇洗净放盘中，吃时将料汁倒入。

9、【煎饼果子】1.将绿豆用搅拌机打成粉，过筛，然后面粉：绿豆粉>4:1的比例混合2.加入牛奶，调成可流动状，把鸡蛋打散加葱花、盐3.在平板锅里刷上薄薄一层油，小火加热，倒入面糊，浇上鸡蛋，两面煎熟4.涂上甜面酱，放上自己喜欢吃的蔬菜，卷起即可。
